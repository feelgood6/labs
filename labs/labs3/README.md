##### Количество бит подсети
Исходная маска у нас 255.255.255.0 новая маска подсети 255.255.255.224 , получается биты нашей подсети можно вычеслить из 224( 128 + 64 + 32) , то есть Количество бит подсети = 3 бита

##### Количество созданных подсетей
Чтобы вычислить количество созданных подсетей 2 возводим в количество бит подсетей , то есть Количество созданных подсетей = 2 * 3 = 8

##### Количество бит узлов в подсети
Чтобы вычеслить количество бит узлов в подсети ,как я понима, нужно взять все 32 бита и вычесть из них количество бит, которые являются маской сети ( в нашем случае 27) , то есть Количество бит узлов в подсети = 32- 27 = 5

##### Количество узлов в подсети
Чтобы вычеслить количество узлов в подсети, нам нужно 2 возвести в степень оставшихся бит и отнять 2 адреса, первый адрес - идентификатор сети, последний - широковещательный, то есть Количество узлов в подсети = 2 * 5 - 2 = 30
